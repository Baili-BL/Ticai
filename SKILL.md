text

🎭 板块情绪量化系统

1. 板块强度（25%）
text

分级：
- >0.6：主线题材（🔥）
- 0.4-0.6：支线题材（⚠️）
- <0.4：轮动题材（📉）

text
监控指标：
- 板块成交额/全A成交额（占比阈值>3%）
- 板块量比均值（理想值1.5-3）
- 大单净流入占比（>0.3为强势）
3. 梯队完整性（15%）
text
检查项（每项满足得分）：
☑ 有3连板以上龙头（0.3分）
☑ 有2连板中军（0.2分）  
☑ 有首板补涨（0.2分）
☑ 有趋势股配合（0.2分）
☑ 有20cm弹性标（0.1分）
总分≥0.8为健康梯队
4. 赚钱效应扩散（15%）
text
监控维度：
- 宽度：涨停股所属细分概念数
- 深度：产业链传导路径
- 热度：相关转债/ETF联动
5. 消息催化强度（10%）
text

一级：国家级政策（权重0.4，有效期3日）

三级：龙头公司公告（权重0.2，有效期1日）
四级：券商集体推荐（权重0.1，有效期当日）
6. 抗跌性测试（10%）
text
测试场景：
1. 大盘下跌时表现
2. 分歧日炸板率
3. 龙头断板后修复

text
认可信号：

启动期（0-30分）：轻仓试错，选辨识度

🛡️ 风控体系
一、三级止损机制
1. 技术止损（第一道防线）
text
- 固定止损：-5%
- 动态止损：跌破10日线
- 时间止损：持有3日不涨即出
2. 板块止损（第二道防线）
text
出现以下信号时减仓：
1. 板块情绪分下降>20分
2. 板块内出现2只以上跌停
3. 龙头炸板且无回封
4. 板块成交占比萎缩30%
3. 市场止损（第三道防线）
text
市场环境恶化时：
- 上涨家数/下跌家数<0.3
- 昨日涨停今日表现<-2%
- 连板高度降至2板以下
→ 仓位降至10%以下
二、动态止盈系统
text
止盈策略：
1. 涨幅10%：上移止损至成本价
2. 涨幅15%：启用5%回落止盈
3. 涨停封板：次日高开无法封板时止盈
4. 板块退潮：无论盈亏立即止盈
三、仓位管理体系
1. 单票仓位计算
python
def calculate_position(stock_score, market_env):
    base_position = 10%  # 基础仓位
    
    # 市场环境调整
    if market_env.赚钱效应 > 0.6:
        base_position += 5%
    if market_env.成交量 > 万亿:
        base_position += 3%
    
    # 个股质量调整
    if stock_score > 80:
        position = min(base_position, 20%)
    elif stock_score > 60:
        position = base_position * 0.7
    else:
        position = base_position * 0.5
        
    return position

text

text

- 9:45后：避免开盘情绪化

- 直线拉升超过5%

1. 涨幅>8%但量能不济

text

3. 震荡环境（无趋势）：
   - 侧重技术突破
   - 降低盈利预期
   - 快进快出
